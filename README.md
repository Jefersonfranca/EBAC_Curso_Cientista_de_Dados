# Entrega de Exercícios do Curso de Ciência de Dados - EBAC

Este repositório contém a entrega dos exercícios do curso de Ciência de Dados oferecido pela [EBAC](https://www.ebac.art.br/). O objetivo é documentar o progresso e compartilhar o aprendizado adquirido ao longo do curso, abordando uma variedade de tópicos essenciais em Ciência de Dados.

## Visão Geral

O curso de Ciência de Dados da EBAC é uma imersão abrangente nos principais conceitos e ferramentas da área. Os exercícios incluídos aqui cobrem desde a introdução à programação e manipulação de dados até técnicas avançadas de machine learning e visualização de dados. Este repositório serve como uma referência pessoal e também como um recurso para outros estudantes que desejam aprender mais sobre Ciência de Dados.

## Estrutura do Repositório

O repositório está organizado por módulos, cada um representando uma etapa do curso. Dentro de cada módulo, há uma pasta específica para cada exercício ou projeto, contendo códigos, notebooks e quaisquer outros arquivos relevantes.

```

├── Módulo_2_Analise_de_Dados/
```

## Como Utilizar

Para explorar os exercícios e projetos, você pode clonar este repositório em sua máquina local e navegar pelos diferentes módulos e exercícios.

### Clone o Repositório

```bash
git clone https://github.com/Jefersonfranca/EBAC_Curso_Cientista_de_Dados.git
```

### Execute os Scripts e Notebooks

Use um ambiente de desenvolvimento como Jupyter Notebook ou um editor de texto para abrir e executar os scripts Python.

## Requisitos

- Python 3.7 ou superior
- Jupyter Notebook
- Pacotes Python: `pandas`, `numpy`, `matplotlib`, `seaborn`, `scikit-learn`, entre outros (veja `requirements.txt` para a lista completa).
